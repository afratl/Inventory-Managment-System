# Inventory Management System

A Django-based inventory management system developed to manage products, categories, brands, firms, purchases, and sales in a structured way.

## Features

* Product listing and detail pages
* Category management
* Brand management
* Firm management
* Purchase records
* Sales records
* Search and filtering functionality
* Stock management
* Admin panel for data management

## Technologies

* Python
* Django
* Django REST Framework
* HTML
* CSS
* JavaScript
* SQLite

## Installation

```bash
git clone https://github.com/afrat1/Inventory-Management-System.git

cd Inventory-Management-System

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```
